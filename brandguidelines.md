# RP Solutionss Brand Guidelines

## Brand Name
**RP Solutionss**

Always written with two "s" at the end. Never written as "RP Solutions" (single s).

### Acceptable Usage
- RP Solutionss
- RP SOLUTIONSS (all caps)

### Unacceptable Usage
- RP Solutions (single s)
- Rp Solutionss (lowercase)
- rp solutionss (all lowercase)
- rp solutionss (no space)
---

## Logo Usage

### Clear Space
Maintain clear space around the logo equal to the height of the "R" in the logo on all sides.

### Minimum Size
- Digital: 120px width minimum
- Print: 1 inch / 25mm width minimum

### Logo Don'ts
- Do not stretch or distort the logo
- Do not rotate the logo
- Do not change the logo colors outside of approved variations
- Do not add effects (shadows, gradients, outlines)
- Do not place the logo on busy backgrounds

---

## Color Palette

### Primary Color

**RP Dark Blue**
- HEX: #2E3A8C
- RGB: 46, 58, 140
- CMYK: 87, 79, 0, 20
- HSL: 232°, 51%, 36%

Use as: Primary brand color, headers, buttons, key UI elements, logo

### Secondary Colors

**RP Light Blue**
- HEX: #4A5FD9
- RGB: 74, 95, 217
- Use as: Accents, hover states, secondary buttons, highlights

**RP Navy**
- HEX: #1A2254
- RGB: 26, 34, 84
- Use as: Dark backgrounds, footer, text on light backgrounds

### Neutral Colors

**White**
- HEX: #FFFFFF
- Use as: Backgrounds, text on dark backgrounds, cards

**Light Gray**
- HEX: #F5F7FA
- Use as: Page backgrounds, subtle dividers

**Dark Gray**
- HEX: #4A4A4A
- Use as: Body text, secondary text

**Black**
- HEX: #1A1A1A
- Use as: Headings, primary text

---

## Typography

### Primary Font
**Inter** (or SF Pro, Helvetica Neue as fallbacks)

### Font Weights
- **Bold (700)**: Headings, emphasis
- **Semi-Bold (600)**: Subheadings, labels
- **Regular (400)**: Body text, paragraphs
- **Light (300)**: Captions, secondary text (use sparingly)

### Font Sizes
- H1: 32px / 2rem
- H2: 24px / 1.5rem
- H3: 20px / 1.25rem
- Body: 16px / 1rem
- Small: 14px / 0.875rem
- Caption: 12px / 0.75rem

### Line Height
- Headings: 1.2
- Body: 1.5

---

## UI Elements

### Buttons

**Primary Button**
- Background: RP Dark Blue (#2E3A8C)
- Text: White (#FFFFFF)
- Border Radius: 8px
- Hover: RP Light Blue (#4A5FD9)

**Secondary Button**
- Background: Transparent
- Border: 2px solid RP Dark Blue (#2E3A8C)
- Text: RP Dark Blue (#2E3A8C)
- Hover: Light blue background tint

### Cards
- Background: White (#FFFFFF)
- Border Radius: 12px
- Shadow: 0 2px 8px rgba(0, 0, 0, 0.08)

### Input Fields
- Border: 1px solid #E0E0E0
- Border Radius: 8px
- Focus Border: RP Dark Blue (#2E3A8C)

---

## Iconography

- Style: Outlined or filled, consistent stroke width
- Size: 24px standard, 20px small, 32px large
- Color: Match text color or RP Dark Blue for emphasis

---

## Tone of Voice

- **Professional**: Clear, trustworthy, knowledgeable
- **Approachable**: Friendly, helpful, not overly formal
- **Concise**: Direct, no jargon, easy to understand
